Desktop взаємодія

  • Double-click на фото — лайк (з анімацією серця)
  • ← → або стрілки на екрані — навігація
  • L — toggle like, D — download, F — favorites only
  • Esc — закрити, повернутись до сітки
  • Click на фото — toggle UI (chrome) приховується для повного перегляду
  • Filmstrip знизу — швидка навігація між сусідніми кадрами

Mobile взаємодія

  • Double-tap — лайк (Instagram-style)
  • Swipe ‹ › — наступне/попереднє фото
  • Swipe ↓ — закрити (drag-to-dismiss)
  • Tap по фото — toggle UI (chrome зникає)
  • Pinch-zoom — масштабування фото
  • Bottom bar: лайк / download / share лінк на це фото
